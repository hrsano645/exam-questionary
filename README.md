## このリポジトリについて

questionayの使い方サンプルです。

## 使い方

* このリポジトリをクローンします。
* pip or ryeを使って、questionaryをインストールします。
* 仮想環境上で `src/exam_questionary/`ディレクトリ内のスクリプトを実行します。
